# X (Twitter) Viral Skill

> Optimize content for X algorithm and engagement patterns.

## When to Use This Skill

Apply when creating:
- Single tweets (280 characters max)
- Threads (numbered sequences)
- Quote tweets
- Reply strategies
- Profile optimization content

## Core Philosophy

X rewards:
1. **Early engagement** - Likes, replies, retweets in first 30-60 minutes
2. **Conversation** - Back-and-forth replies
3. **Dwell time** - People stop to read/watch
4. **Native content** - No external links in main tweet

X punishes:
1. External links in first tweet (kills reach 30-50%)
2. Engagement bait ("Like if you agree")
3. Over-posting (more than 3-5x/day)
4. Editing tweets within first hour

## Thread Best Practices

### Opening Tweet Requirements
1. Must stand alone (people see it first)
2. Creates curiosity or makes a bold claim
3. Signals thread with "ðŸ§µ" or "Thread:" or "ðŸ‘‡"
4. No links (save for last tweet or reply)

### Middle Tweets
- Number them (3/, 4/, etc.) for easy following
- Each tweet should deliver value alone
- Use callbacks ("Remember point 2?")
- Vary lengths (short punchy + longer detail)

### Closing Tweet
- Summarize key insight
- Clear CTA
- "Follow @handle for more [topic]"
- Can include link here

### Thread Length
| Type | Tweets | Best For |
|------|--------|----------|
| Quick hit | 3-5 | Single insight, hot take |
| Standard | 6-10 | Framework, story, breakdown |
| Deep dive | 10-15 | Comprehensive guide |

## Quote Tweet Strategy

### When to Quote Tweet
- Adding unique perspective to someone else's point
- Disagreeing respectfully with reasoning
- Amplifying content with personal experience
- Turning someone's question into content

### Quote Tweet Structure
```
[Your take or addition]

[Why it matters or what's missing]

[Optional: question for engagement]
```

### Quote Tweet Don'ts
- Don't just say "This" or "So true"
- Don't quote without adding value
- Don't use to argue (reply instead)

---

## Reply Strategy

### Replies That Build Followers
1. **Add value** - Share relevant experience
2. **Ask thoughtful questions** - Not generic
3. **Respectfully disagree** - With reasoning
4. **Share resources** - Links in replies are fine

### Reply Structure
```
[Acknowledge their point briefly]

[Add your unique perspective or data]

[Question or call to continue conversation]
```

### Accounts to Reply To
- Your ideal customers/audience
- Larger accounts in your space
- Trending topics in your niche
- Complementary products/services

## Algorithm Tips

### First Hour Critical
- Post when your audience is online
- Reply to every comment quickly
- Like and engage with responses
- Don't edit the tweet

### Boost Signals
- Replies > Likes (conversation wins)
- Bookmarks indicate high value
- Retweets extend reach
- Profile clicks = curiosity generated

### Posting Schedule

**Best Times (General):**
- Weekdays: 8-10am, 12-1pm, 5-7pm
- Weekends: 9-11am
- Avoid: Late night, early morning

**Frequency:**
- Minimum: 1-2x/day for growth
- Optimal: 3-5x/day
- Threads: 1-2x/week

**Pattern:**
```
Morning: Insight or hot take
Midday: Thread or value content
Evening: Engagement (replies, QTs)
```

## Hashtag Strategy

- Use 0-2 hashtags (often 0 performs better)
- If using: 1 broad (#AI), 1 niche (#NoCode)
- Don't put hashtags mid-sentence
- Only relevant, non-spammy tags

### Engagement Rate
- Good: 2-5%

- Viral: 10%+

### Profile Clicks
High profile clicks = curiosity generated = good hooks

### Bookmark Rate
Bookmarks = high-value content worth saving

### Reply Ratio
More replies than likes = conversation content (best for growth)
