# X (Twitter) Post Template

## Format Rules
- **Length:** 280 characters max
- **Thread:** Use for longer announcements (4-6 tweets)
- **Hook:** First tweet must standalone and grab attention
- **Media:** Include image/video when possible

## Single Tweet Structure

```
[HOOK + VALUE in one punch]

[LINK or CTA]
```
