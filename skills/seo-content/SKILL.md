# SEO Content Skill

> Rank on Google search through strategic, search-optimized content.

## When to Use This Skill

Apply when creating:
- Blog posts targeting search traffic
- Pillar pages and content hubs
- How-to guides and tutorials
- Product comparison pages
- FAQ pages
- Glossary/definition pages

## Core Philosophy

**SEO content = answering questions better than anyone else.**

Google rewards:
1. **Search intent match** - Content answers what searcher actually wants
2. **Depth** - Comprehensive coverage of the topic
3. **E-E-A-T** - Experience, Expertise, Authoritativeness, Trust
4. **User signals** - Low bounce, high dwell time, engagement

## Keyword Strategy

### Primary Keyword Placement

| Location | Priority | Example |
|----------|----------|---------|
| Title (H1) | Required | "AI Implementation Guide: 7 Steps to Success" |
| URL slug | Required | `/ai-implementation-guide` |
| First 100 words | Required | Mention naturally in intro |
| Meta description | Required | Include keyword + CTA |
| H2 headings | 2-3 times | Use variations |
| Image alt text | 1-2 times | Describe image + keyword |

### Keyword Density
- Target: 1-2% (natural usage)
- Avoid: Keyword stuffing (>3%)
- Use: Semantic variations and related terms

### Long-Tail Keywords
Include 3-5 related long-tail keywords throughout:
```
Primary: "AI implementation"
Long-tail:
- "how to implement AI in small business"
- "AI implementation challenges"
- "AI implementation cost"
- "AI implementation timeline"
```

## Content Structure

### Standard Article Structure

```
[Title - H1] (Include primary keyword)

[Meta description] (150-160 chars, keyword + benefit + CTA)

[Featured image with keyword-rich alt text]

[Introduction - 100-150 words]
- Hook the reader
- State the problem
- Promise the solution
- Include primary keyword

[Table of Contents] (for posts 2000+ words)

[H2: First Major Section]
[H3: Subsection if needed]
Content...

[H2: Second Major Section]
Content...

[...continue sections...]

[H2: FAQ Section] (3-5 questions)
Q: [Common question with keyword]
A: [Direct answer, then expansion]

[H2: Conclusion]
- Summarize key points
- Clear CTA
- Internal links to related content

[Author bio with credentials]
```

### H2/H3 Heading Rules
- Every H2 should be a potential search query
- Use H3 for sub-topics within H2
- Include keywords naturally
- Make scannable (readers skim headings)

## Featured Snippet Optimization

### Paragraph Snippets (Definition)
```
**What is [keyword]?**

[Keyword] is [direct 40-60 word definition that answers the query completely].
This includes [2-3 key components]. For example, [brief example].
```

### List Snippets
```
**How to [keyword]:**

1. **[Step 1]** - Brief explanation
2. **[Step 2]** - Brief explanation
3. **[Step 3]** - Brief explanation
...
```

### Table Snippets
```
| Feature | Option A | Option B |
|---------|----------|----------|
| Price | $X | $Y |
| Best for | Use case | Use case |
```

## On-Page SEO Checklist

### Technical
- [ ] URL is short and includes keyword
- [ ] Page loads in <3 seconds
- [ ] Mobile responsive
- [ ] HTTPS enabled
- [ ] Schema markup added (Article, FAQ, HowTo)

### Content
- [ ] Title includes primary keyword (front-loaded)
- [ ] H1 matches title tag
- [ ] Keyword in first 100 words
- [ ] 2-3 H2s include keyword variations
- [ ] Images have descriptive alt text
- [ ] Internal links to related content (3-5)
- [ ] External links to authoritative sources (2-3)

### User Experience
- [ ] Clear, scannable structure
- [ ] Short paragraphs (3-4 sentences max)
- [ ] Bullet points and lists where appropriate
- [ ] Visual breaks (images, callouts, quotes)
- [ ] Table of contents for long content

## Search Intent Matching

### Informational Intent
*Searcher wants to learn*

Keywords: "what is", "how to", "guide", "examples"
Content: Educational, comprehensive, visual aids
CTA: Newsletter signup, free resource

### Commercial Intent
*Searcher is evaluating options*

Keywords: "best", "top", "vs", "comparison", "review"
Content: Objective analysis, pros/cons, recommendations
CTA: Free trial, demo, consultation

### Transactional Intent
*Searcher is ready to act*

Keywords: "buy", "pricing", "discount", "near me"
Content: Clear value prop, pricing, testimonials
CTA: Purchase, signup, contact

### Navigational Intent
*Searcher looking for specific page*

Keywords: Brand names, product names
Content: Landing pages, product pages
CTA: Direct to destination

## Internal Linking Strategy

### Hub & Spoke Model
```
[Pillar Page: AI Implementation Guide]
    ├── [Spoke: How to Choose AI Vendor]
    ├── [Spoke: AI Implementation Costs]
    ├── [Spoke: AI Implementation Timeline]
    ├── [Spoke: Common AI Mistakes]
    └── [Spoke: AI Success Stories]
```

### Linking Rules
- Every post links to pillar page
- Every post links to 2-3 related spokes
- Use keyword-rich anchor text (not "click here")
- Link to high-value pages from high-traffic pages

## Content Refresh Strategy

### When to Update
- Rankings dropping
- Information outdated
- Competitors outranking
- 6+ months since last update

### What to Update
- Add new sections for new subtopics
- Update statistics and examples
- Add new internal links
- Improve featured snippet formatting
- Expand thin sections
- Update publish date (if substantial changes)

## Writing for SEO + Humans

### Balance Rules
1. **Write for humans first** - Engaging > optimized
2. **Add keywords naturally** - If it sounds forced, rewrite
3. **Answer the question fast** - Don't bury the lead
4. **Provide unique value** - Say something competitors don't
5. **Use original examples** - Generic = forgettable

### Quality Signals
- Original research or data
- Expert quotes or interviews
- Real-world examples with specifics
- Author expertise visible (bio, credentials)
- Regular updates (freshness)

## Meta Description Template

```
[Primary keyword] explained: [Main benefit]. Learn [what they'll learn]
with [unique value prop]. [CTA] →

Example:
AI implementation guide for 2024: Get your team using AI in 14 days,
not 6 months. Proven 7-step framework with templates. Start free →
```

Character limit: 150-160 (Google truncates longer)

## Schema Markup Types

### Article Schema
```json
{
  "@type": "Article",
  "headline": "Title",
  "author": {"@type": "Person", "name": "Ofer Blutrich"},
  "datePublished": "2024-01-15",
  "dateModified": "2024-01-20"
}
```

### FAQ Schema
```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Question text?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Answer text"
    }
  }]
}
```

### HowTo Schema
For step-by-step guides - enables rich results in search.
