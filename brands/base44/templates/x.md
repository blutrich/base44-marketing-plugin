# X (Twitter) Post Template

## Format Rules
- **Length:** 280 characters max (single tweet)
- **Thread:** Use for longer announcements (5-12 tweets)
- **Hook:** First tweet must standalone and grab attention
- **Media:** Include image/video when possible
- **Links:** Put in reply, not main tweet

```
[Bold observation or claim]

[Quick proof or context]

```

Build something simple that works. That's it.
